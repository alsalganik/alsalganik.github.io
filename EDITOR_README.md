# Site Editor — Local Content Management

Два удобных локальных редактора для управления контентом сайта без редактирования HTML кода.

## 1️⃣ Визуальный HTML редактор (Рекомендуется)

Красивый графический интерфейс в браузере.

### Запуск:

```bash
python3 scripts/editor_server.py
```

Затем откройте в браузере: **http://localhost:8001**

### Особенности:

- 🎨 Красивый визуальный интерфейс
- 📱 Адаптивный дизайн
- ✨ Предпросмотр топиков в реальном времени
- 💾 Мгновенное сохранение
- 🔒 Работает только локально (не в интернете)

---

## 2️⃣ Командная строка (CLI редактор)

Простой текстовый интерфейс в терминале.

### Запуск:

```bash
python3 scripts/edit_site.py
```

### Меню опций:

1. **Profile Title** — главный заголовок на главной странице
2. **Bio** — краткое описание профиля
3. **Research Overview** — подробное описание исследований
4. **Research Topics** — теги исследований (через запятую)
5. **Grants Information** — информация о грантах (страница CV)

### Как редактировать:

1. Запустите скрипт
2. Выберите пункт меню (1-5)
3. Прочитайте текущее содержимое
4. Введите новый текст
5. Нажмите Enter два раза, чтобы сохранить

### Пример:

```
Select option (0-5): 1

📝 Profile Title (Main headline)
Current: X-ray pulsars, high-energy astrophysics
Enter new text (press Enter twice to finish):
My Research Topic
[Enter]
[Enter]
✓ Profile title updated!
```

---

## Безопасность

- ✅ Оба редактора работают **только локально** на вашем компьютере
- ✅ **Не доступны** из интернета
- ✅ Безопасно редактируют только текстовое содержимое
- ✅ HTML структура остаётся нетронутой
- ✅ Все изменения сохраняются в `/index.html` и `/cv/index.html`
